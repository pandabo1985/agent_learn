# UI 设计规范

> 基于《AI 概念关系讲解》HTML 页面（浅色主题版）提炼，适用于同类知识讲解类页面。

---

## 一、配色系统

| 变量名 | 色值 | 用途 |
|---|---|---|
| `--bg` | `#f5f6fa` | 页面背景 |
| `--sidebar-bg` / `--content-bg` | `#ffffff` | 侧边栏 / 卡片背景 |
| `--border` | `#e2e6ed` | 边框、分割线 |
| `--text` | `#2c3e50` | 正文主色 |
| `--text-muted` | `#7f8c9a` | 次要文字、未激活导航 |
| `--accent` | `#4f6ef7` | 主强调色（蓝紫） |
| `--accent-light` | `#eef1fe` | 强调色浅底（callout / hover） |
| `--accent-hover` | `#3a56e0` | 强调色深版（hover 状态） |
| `--heading` | `#1a2540` | 标题专用深色 |
| `--code-bg` | `#f0f2f8` | 代码块背景 |
| `--tag-bg` / `--tag-text` | `#e8ecfd` / `#3a56e0` | Chip 标签 |
| `--timeline-line` | `#c5cde8` | 时间线竖线 |
| `--shadow` | `0 2px 12px rgba(79,110,247,0.07)` | 卡片投影 |

**原则**：白底 + 米白页面背景，单一蓝紫强调色，保持低饱和、高可读性。

---

## 二、布局结构

- **双栏布局**：左侧固定侧边栏 `240px` + 右侧内容区
- 侧边栏：`position: fixed`，独立滚动，右侧 `1px border` 分隔
- 内容区：`margin-left: 240px`，最大宽度 `900px`，内边距 `48px 52px`
- 移动端（`≤768px`）：隐藏侧边栏，内容区满宽

---

## 三、排版规范

| 元素 | 规格 |
|---|---|
| 基础字体 | `-apple-system, BlinkMacSystemFont, 'Segoe UI'`，`15px / 1.7` |
| 等宽字体 | `'SFMono-Regular', Consolas`，用于层级图、代码 |
| 页面大标题 `h1` | `2rem / 800`，颜色 `--heading`，关键词用 `--accent` |
| 章节标题 `h2` | `1.25rem / 700`，颜色 `--heading` |
| 章节编号 `.sec-num` | `44×44px` 方形框，`2px solid --accent`，圆角 `8px` |

## 四、组件模式

### 4.1 侧边栏导航
- 顶部 LOGO 区：大写字母 + `--accent` 颜色 + 底部分割线
- 链接：`13px`，默认 `--text-muted`，`3px` 透明左边框
- 激活态：左边框变 `--accent`，背景 `--accent-light`，文字 `--accent`，`font-weight: 600`
- 滚动高亮：`IntersectionObserver` 监听 section，`threshold: 0.4`

### 4.2 卡片 `.card`
- 白色背景 + `1px --border` + `border-radius: 10px` + `--shadow`
- 内边距：`20px 24px`，卡片间距 `16px`

### 4.3 Callout 提示块
- 背景 `--callout-bg`（`--accent-light`），左侧 `4px solid --accent`
- 圆角右侧：`0 8px 8px 0`，`14px` 字号
- 用于「一句话总结」或核心结论

### 4.4 Chip 标签
- 背景 `--tag-bg`，文字 `--tag-text`，圆角 `20px`，`3px 12px` 内边距
- `12px / 600 / letter-spacing .03em`，横向 `flex-wrap` 排列

### 4.5 层级图 `.hierarchy`
- 等宽字体，`line-height: 2`
- 颜色分层：root `--accent`，L1 `#2d6a4f`（绿），L2 `--text-muted`，L3 `#a0aab8`
- 每层用 `padding-left` 阶梯缩进（`20 / 44 / 68px`）

### 4.6 时间线 `.timeline`
- 左侧伪元素竖线 `2px`，颜色 `--timeline-line`
- 节点：`10×10px` 圆形，`--accent` 填充，外环 `2px solid --accent`
- 年份标签：`11px / 700 / --accent / letter-spacing .06em`

### 4.7 对比表格
- 表头：`--accent` 背景 + 白字，`font-weight: 600`
- 隔行：偶数行 `--accent-light` 底色
- `border-radius: 10px`，`overflow: hidden` 包裹，加 `--shadow`

---

## 五、交互规范

- 链接 hover：背景 `--accent-light`，文字 `--accent`，过渡 `0.18s`
- 滚动条：宽度 `5px`，滑块颜色 `--border`，圆角 `4px`，轨道透明
- 章节滚动偏移：`scroll-margin-top: 32px`（避免被固定侧边栏遮挡）
