02

机器学习 & NLP 的关系

监督学习无监督学习 文本分类翻译问答

机器学习（ML）

让计算机从数据中自动学习规律，不需要完全手写规则。它是一个广义方法体系，包含决策树、SVM、神经网络等众多算法。

NLP 与 ML 的关系

NLP（自然语言处理）是 AI 中处理人类语言的应用方向。早期 NLP 也使用规则和语言学方法；现代 NLP 的主流实现主要依赖 ML，尤其是深度学习、Transformer 和 LLM。

类比：ML 是一类“学习方法”，NLP 是“语言问题领域”；当用 ML 去解决翻译、摘要、问答等语言问题时，就形成了现代 NLP 的主流路线。
03

机器学习与深度学习的关系

从数据中学习多层神经网络特征学习

一句话理解

深度学习是机器学习的一种方法，而且是目前最重要的一类机器学习方法。机器学习是大类，深度学习是其中基于多层神经网络的分支。

人工智能 AI
└─ 机器学习 ML：让机器从数据中学习规律
   ├─ 传统机器学习：线性回归 / 决策树 / 随机森林 / SVM / KNN
   └─ 深度学习 DL：用多层神经网络进行机器学习
      ├─ CNN
      ├─ RNN / LSTM / GRU
      └─ Transformer / 主流 LLM
对比机器学习 ML深度学习 DL
范围更大的方法体系机器学习的一个子集
核心方法多种算法：决策树、SVM、随机森林、回归等多层神经网络
特征处理经常需要人工设计特征通常自动从数据中学习特征
数据与算力小数据也常可使用，算力要求相对低通常需要更多数据和更强算力
擅长任务表格数据、分类、预测、简单决策图像、语音、自然语言、生成式 AI
直观例子：判断一张图片是不是猫。传统机器学习通常需要人先设计“耳朵、眼睛、毛发”等特征；深度学习可以直接从大量图片中自动学习“边缘 → 纹理 → 局部结构 → 猫脸 → 猫”的层级特征。
最简单总结：机器学习 = 让机器从数据中学习；深度学习 = 用很多层神经网络来做机器学习。所以，深度学习属于机器学习。
04

深度学习家族：CNN / RNN / Transformer

卷积神经网络循环神经网络注意力机制
架构核心思想擅长
CNN用卷积核扫描局部特征图像、局部模式识别、部分短文本分类
RNN / LSTM / GRU按顺序逐步处理，并把“记忆”传递到下一步时序、语音、早期 NLP
Transformer训练时高度并行，并用自注意力建模全局关系翻译、生成、长距离依赖建模、主流 LLM
关键差异：CNN 更关注局部模式，RNN 按顺序传递记忆，Transformer 用自注意力直接计算任意位置之间的关系，并在训练阶段更容易并行。
05

RNN 系列：RNN → LSTM → GRU

基础版 · RNN

逐词或逐步处理，把上一步的“隐状态”传给下一步。缺点是长序列中早期信息容易逐渐衰减，并受到梯度消失问题影响。

升级版 · LSTM

增加“记忆细胞”和三个门控（输入门 / 遗忘门 / 输出门），可以选择性地记住或遗忘信息，显著缓解长距离依赖和梯度消失问题。

精简版 · GRU

把 LSTM 的门控结构简化为两个门（重置门 / 更新门），参数更少，训练通常更快，在很多任务上效果与 LSTM 接近。

三者关系：它们属于同一家族，核心都是“顺序处理 + 状态传递”。LSTM 和 GRU 是对基础 RNN 的改进，不是完全不同的路线。

Transformer：为什么能取代 RNN？

Self-Attention训练并行 位置编码Encoder-Decoder
维度RNN / LSTMTransformer
计算方式逐步串行，必须等待上一步状态训练时多个位置可同时计算，GPU 利用率更高 ✅
长距离依赖靠状态和门控维持，距离越远越容易衰减自注意力可以直接连接任意两个位置 ✅
训练速度串行依赖强，难以充分并行训练阶段高度并行，适合大规模数据训练 ✅
超长文本序列很长时效果下降明显更擅长长距离依赖；但标准注意力复杂度高，超长文本成本较大
核心突破 = 训练并行 + 自注意力。
并行让训练速度大幅提升；自注意力让模型不必只靠“记忆传递”，而是直接计算词与词、句与句之间的关联。注意：自回归生成时通常仍要一个 token 一个 token 生成，并不是推理阶段完全并行。

LLM：基于 Transformer 的大规模语言模型

GPT 系列BERT / T5预训练 + 微调对齐

什么是 LLM？

Large Language Model，通常指在海量文本、代码或多模态数据上预训练的大规模语言模型。当前主流 LLM 多基于 Transformer 或其变体，参数量可从数十亿到数千亿级。

LLM 与 Transformer 的关系

主流 LLM 通常基于 Transformer 或 Transformer 变体，但不能简单说“LLM 就是 Transformer”。LLM 的能力不仅来自架构放大，还来自高质量数据、预训练目标、指令微调、偏好对齐和推理优化。

LLM 与 NLP 的关系

LLM 是当前 NLP 中最通用、最强大的实现方式之一。传统 NLP 往往针对每个任务训练专门模型；LLM 则能用一个通用基础模型完成翻译、摘要、问答、信息抽取、对话等多种任务。

一句话：LLM = Transformer/变体 × 大规模参数 × 高质量数据 × 预训练/微调/对齐。它是当前 NLP 的主流范式，但不是唯一终点。
